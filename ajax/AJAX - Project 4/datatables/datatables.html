Tables Plugin

DataTables is a plugin for working with tables allowing you to sort by column, paginate data, add themes, and search! You simply include the css and js CDN provided on the datatables website, and follow the instructions on the website. The short version of those instructions is:

  • link the css and js CDNs
  • datatables call to table in js ready function
    • $(document).ready(function(){
    $('#myTable').DataTable();
});      
          
    • <table id="myTable" class="display" cellspacing="0" width="100%">
  • utilize thead, tfoot, and tbody
