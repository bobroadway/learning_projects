Part 2 Problem Statement

Write a program that asks a user to enter a number and a letter. If the number is less than 5 and the letter is A or B, then display on the page a message showing the inputs and the text, are correct. Otherwise, display the inputs and a message that says, "<entered number> and/or <entered letter> are not correct, please try again".

Part 2 Program Planning Lists

  • In the following pre tag, write the three planning lists based on the verbs and nouns you see in the problem statement.
The List of Input Variables:
    enteredNumber
    enteredLetter

The List of Output Variables:
    outputMessage

The Process Checklist:
    Declare Variables
    Prompt the user for a number and a letter, begin the output message
    Find if the number is less than 5, find if the number is A or B
    Display outputMessage
