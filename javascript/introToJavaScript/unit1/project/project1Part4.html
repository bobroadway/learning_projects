Problem Statement

Write a program that receives, via the prompt() method, five strings into variables. The program calculates the length of each string and stores the length into variables. For each entered string the program will output a line like the sample output below. (Remember that the sample strings shown in the sample output below are just samples--the program must be able to accept any five strings keyed in and display those keyed strings).

Hint: Review this if you've forgotten how to get the length of a string: The length property of a string object
